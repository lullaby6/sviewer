# Sviewer

Image Viewer built with Vue3 and Tailwind CSS.

[Go to Sviewer](https://sviewer.vercel.app/)

## Features

- PWA
- Zoom In & Zoom Out
- Rotate
- Fullscreen
- Open Multiple Images
- Pixel Art Rendering
- Copy Image to Clipboard
- Donwload Image
- Open Image from Clipboard
- Open Image from URL
- Draggeable Image
- Hotkeys
